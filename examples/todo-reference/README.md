# Todo Reference Fixture

This runnable fixture demonstrates the Todo reference project's existing task flow at desktop and mobile widths. It is a QA aid, not a baseline-vs-rules benchmark result and not evidence for changing benchmark scores.

## Covered Behaviour

- today-first task list and quick capture;
- textual error recovery for an empty submission;
- an asynchronous save state without a layout jump;
- completion state;
- detail inspection that keeps the list visible on desktop and behind a mobile sheet;
- visible keyboard focus, accessible names, 44px controls, and reduced-motion handling.

## Run Locally

```bash
python3 -m http.server 4173 --directory examples/todo-reference
```

Open `http://127.0.0.1:4173` and use the Playwright CLI for a reproducible check:

```bash
export PWCLI="$HOME/.codex/skills/playwright/scripts/playwright_cli.sh"
"$PWCLI" --session todo-reference open http://127.0.0.1:4173
"$PWCLI" --session todo-reference resize 390 844
"$PWCLI" --session todo-reference snapshot
```
