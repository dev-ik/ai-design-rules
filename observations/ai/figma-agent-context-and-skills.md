# Design Agents Improve Through Real Context And Reusable Skills

## Source

Figma's 2026 design agent and Config announcements.

## Observed Behavior

Figma lets the design agent use file structure, components, tokens, attachments, web research, connectors, and reusable team skills. Prompt history is also treated as shared process context rather than only disposable chat.

## Why It May Matter

Prompt wording alone cannot reliably carry product identity. Agents need scoped project evidence, reusable design decisions, and direct access to the system they are expected to apply.
