# Notes App Benchmark

## Product Brief

Create a modern consumer notes app for quick personal capture and retrieval.

The app should help a user create a note quickly, find recent notes, inspect note details, and recover from failed save or sync states.

## Expected User

- Consumer capturing thoughts, lists, snippets, and reminders.
- Uses mobile for quick capture and desktop for review.

## Expected Platform

- Responsive web app.
- Mobile-first behavior required.

## Constraints

- No team workspace.
- No publishing flow.
- No complex folder management required.
- Include empty, loading, and error states.
- Preserve note content during failures.

## Evaluation Criteria

- Product Thinking
- UX
- Information Architecture
- Navigation
- Accessibility
- Mobile-first
- State Design
- Simplicity
- Overall Product Quality
