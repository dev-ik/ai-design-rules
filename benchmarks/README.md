# Benchmarks

Benchmark scenarios define reproducible product briefs for comparing:

```text
Baseline AI -> AI + AI Design Rules
```

Do not add implementation code here.

Each scenario should include:

- product brief;
- constraints;
- expected user;
- expected platform;
- evaluation criteria.

Use `docs/BENCHMARK.md` for procedure and `docs/EVALUATION_RUBRIC.md` for scoring.
